# SC-24 アイデアチャット（画面設計 / 機能詳細）

> 画面遷移図: `doc/画面設計/画面遷移図.md` ／ モック: `doc/画面設計/mocks/SC-24_アイデアチャット.html` ／ 要件: `doc/要件定義/README.md`（チャット・添付・メンション・通知）
> 対象デザイン標準: `doc/画面設計/デザイン標準.md`（業務層＝クリーン）

- 最終更新: 2026-07-18
- ステータス: ドラフト（機能詳細）

## 1. 概要・目的

- **1つのアイデアに紐づくチャット**（Idea ⇔ ChatGroup が 1:1）。パーティー内で議論・ブラッシュアップする場。
- 表示は **フラットリスト（Slack/Teams 風）**＝アバター＋氏名＋本文を横幅で並べる（長文・メンション・添付が読みやすい・SC-22 プレビューと一貫）。
- SC-22 の「チャットを開く」／通知（SC-02）から遷移。**フルページ**（モーダルにはしない）。

## 2. アクセス条件・権限

| 項目 | 内容 |
| --- | --- |
| 閲覧 | **当該クエストのパーティー全員**（アイデアの公開範囲＝パーティー内） |
| 投稿 | **コメント作成権限**（新規参加メンバーに既定付与）。無い場合は入力欄を無効化＋理由表示 |
| 添付 | 投稿権限に含む。実体は MinIO・物理名ハッシュ化・元名は DB 保持 |
| メンション | パーティーメンバーを `@` で指定 → 対象にアプリ内通知 |

## 3. レイアウト構成（決定）

- **上部＝コンパクトな文脈バー**（`.chat-context`）: クエスト名・カテゴリ／アイデア件名／参加人数・件数／「アイデア詳細を開く（SC-22）」／← 戻る。
- **中央＝メッセージスレッド**（`.chat-thread`・フラットリスト）: 日付セパレータ・未読セパレータ・メッセージ（アバター＋氏名＋時刻＋本文＋添付）。
- **下部＝入力欄**（`.composer`・`position: sticky; bottom:0`）: テキストエリア（自動リサイズ）＋ 📎添付 ＋ 送信、`@` メンション候補ポップアップ、補足（Enter送信/Shift+Enter改行・権限・+5XP）。

## 4. 表示項目

### 4.1 文脈バー
| 項目 | 補足 |
| --- | --- |
| クエスト名・カテゴリ | 小さめ（文脈） |
| アイデア件名 | 💬＋件名（1行省略） |
| メタ | 参加人数・コメント件数 |
| アイデア詳細（SC-22） | ボタンで戻る先を明示 |
| ← 戻る | SC-22（呼び出し元） |

### 4.2 メッセージスレッド
| 項目 | 種別 | 補足 |
| --- | --- | --- |
| 日付セパレータ | `.chat-day` | 日付が変わる区切り |
| 未読セパレータ | `.unread-sep` | 「ここから未読」。最終既読位置に表示 |
| アバター/氏名 | `.avatar sm` ＋名前 | 自分の投稿は「（あなた）」タグ |
| 時刻 | テキスト | HH:MM |
| 本文 | テキスト | 改行・長文対応。`@氏名` を強調（**自分宛メンションは警告色でさらに強調**） |
| 添付 | `.file-chip` | ファイル名＋サイズ＋ダウンロード（署名付きURL） |

### 4.3b リアクション（通常＋魔法）
- 各メッセージ下部に**リアクションバー**（`.reaction-bar`）。**＋**（`.reaction-add`）からピッカーを開く。
- **2層**: ①**通常リアクション**（絵文字 👍❤️😄🎉🙏👀・誰でも無料・エフェクトなし）②**魔法リアクション**（SC-32 で SP 解放済みの魔法・**エフェクトあり・クールダウンあり**）。
- リアクションは**集計チップ**（絵文字/アイコン＋数）。自分が付けた分は `.is-me` で強調。魔法チップは `.magic`（レア色の光る縁）。
- **魔法エフェクト**は対象メッセージ枠に**永続付与**（例: 炎＝`spell-fx--fire` で枠が燃える。ほか 氷/雷/キラキラ）。`prefers-reduced-motion` で動きを止め静的な色味のみ。
- **クールダウン**: 使用後、その魔法はピッカーで一定時間 disabled（残り秒を表示）。解放は一度きり（SP）なので使用自体は無料。
- リアクション・魔法は**装飾/社交演出のみ**（XP・評価・投票に非影響）。

### 4.3 入力欄（composer）
| 項目 | 補足 |
| --- | --- |
| テキストエリア | 自動リサイズ（最大 180px）。`@` 入力でメンション候補 |
| メンション候補 | パーティーメンバーを部分一致で絞り込み。↑↓/Enter/Tab で選択・Esc で閉じる |
| 📎 添付 | 添付予定をチップ表示（×で取消）。複数可 |
| 送信 | Enter 送信 / Shift+Enter 改行 |
| 補足 | 権限・投稿で +5 XP（日次上限）・添付/メンション/通知対応 |

## 5. アクション・遷移

| トリガー | 条件 | 遷移/結果 |
| --- | --- | --- |
| 送信 | コメント権限 | メッセージ追加・+5 XP（日次上限）・メンション対象へ通知・末尾へスクロール |
| `@` 入力 | — | メンション候補ポップアップ（メンバー絞り込み） |
| 📎 添付 | コメント権限 | ファイル選択（MinIO へアップロード・権限検証） |
| 添付ダウンロード | パーティー | 署名付きURLで取得 |
| アイデア詳細を開く / ← 戻る | — | **SC-22 アイデア詳細** |
| メンション/通知から流入 | — | 該当メッセージ位置へスクロール（SC-02 通知経由） |
| ＋ リアクション | パーティー | ピッカー表示（通常絵文字＋解放済み魔法） |
| 通常リアクション選択 | パーティー | 集計チップを増減（自分の分をトグル） |
| 魔法リアクション発動 | 魔法を解放済み・クールダウン外 | メッセージ枠にエフェクト付与＋魔法チップ＋クールダウン開始。**魔法の解放は SC-32** |

## 6. 状態・バリエーション

| 状態 | 表示 |
| --- | --- |
| コメント権限なし | 入力欄を無効化＋「投稿するにはコメント権限が必要です」 |
| 未読あり | 未読セパレータを表示・初期スクロールを未読先頭へ |
| 0件 | 「まだコメントはありません。最初のコメントを投稿しましょう」 |
| 締切/クローズ後 | 投稿可否はステータス＋サーバー判定（不可時は入力欄無効化＋理由）→ TBD |
| 自分の投稿 | 「（あなた）」タグ。編集/削除の可否は TBD |

## 7. データ・API（想定）

- **GET `/api/ideas/{id}/chat?before=&limit=`**: メッセージ一覧（投稿者/本文/時刻/添付/メンション）。ページング（過去へ遡上）。既読位置。
- **POST `/api/ideas/{id}/chat`** `{text, attachments[], mentions[]}`: 投稿。+5 XP（初回・日次上限）。メンション対象へ Notification 生成。
- **POST `/api/uploads`**（共通）: 添付アップロード（MinIO・ハッシュ名・元名/サイズ/MIME 保存）。
- **POST `/api/ideas/{id}/chat/read`**: 既読位置更新（未読セパレータ用）。
- 将来: リアルタイム配信（WebSocket / SSE）。MVP はポーリング/再取得でも可（TBD）。

## 8. アクセシビリティ / 表示配慮

- 入力欄・候補は role/aria（`listbox`/`option`・`aria-modal` 不要）。候補はキーボード操作対応。
- メンションは色のみに依存しない（`@` 記号＋文字装飾）。
- `prefers-reduced-motion` 時はスクロールアニメーションを抑制。
