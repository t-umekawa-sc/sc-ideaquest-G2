# SC-41 ランキング（画面設計 / 機能詳細）

> 画面遷移図: `doc/画面設計/画面遷移図.md` ／ モック: `doc/画面設計/mocks/SC-41_ランキング.html` ／ 要件: `doc/要件定義/README.md`（FR-10・第6節・第10節ランキング）
> 対象デザイン標準: `doc/画面設計/デザイン標準.md`（**ゲーム層**＝レトロ/ドット・CRTガラス）

- 最終更新: 2026-07-19
- ステータス: ドラフト（機能詳細）

## 1. 概要・目的

- **会社内（テナント内）の全メンバーのランキングを全件表示**する画面（ゲーム層）。ダッシュボード（SC-01）の週間ランキング TOP3 の「すべて見る」の遷移先。
- **スコア＝期間内の獲得XP＋獲得コインの単純合算**（既存の週間ランキングと同定義）。**期間を切替**（今週 / 先週 / 今月 / 通算）して順位付け。
- 導線: **ダッシュボード（SC-01）の週間ランキングパネル「ランキングをすべて見る →」**／ショートカットタイル。
- クエスト単位の順位は **SC-12「クエスト内 週間ランキング」**（別軸）。本画面は**全社**。

## 2. アクセス条件・権限

| 項目 | 内容 |
| --- | --- |
| アクセス可能者 | 認証済みユーザー全員（自社メンバーのランキングを閲覧） |
| 対象範囲 | **同一会社（テナント）内の全ユーザー**。他社は非表示（会社DBで自然に分離） |
| スコア | 期間内の **獲得XP＋獲得コイン**（`Transaction/Activity` を期間集計）。**SP はランキング対象外**・魔法/リアクションは非影響 |

## 3. レイアウト構成（決定・ゲーム層）

- **共通ヘッダー**（Lv/コイン/SP＋ベル＋アバターメニュー）。
- **期間切替（クリーンなセグメント `.rank-tabs`）**: 今週（既定）/ 先週 / 今月 / 通算。集計範囲の注記を併記。
- **あなたの順位サマリー（`.card.card-accent`・プライマリ帯）**: アバター＋氏名＋**現在の順位（N位 / 全M人中）**＋スコア＋**「▼ 自分の順位へ」**（全件リストの自分の行へスクロール）。
- **ランキング本体（`.pixel-panel.rank-panel`＝CRTガラス）**:
  - **表彰台（`.podium`）**: TOP3 を 2位・1位・3位の順に並べ、1位を最も高く（メダル＋アバター＋スコア＋順位ブロック）。
  - **全件リスト（`.rank-list`・全社員）**: 順位番号＋メダル（TOP3）＋アバター＋氏名＋スコア（合計＋EXP/コイン内訳）。**自分の行は `.is-me` でハイライト**。
- **レイヤー方針（1レイヤー1マテリアル）**: ランキング本体は CRTガラス（ゲーム層の顔＝`.rank-panel` を全件で流用）。期間切替・自分の順位サマリーはページUI（クリーン）。SC-01 週間ランキングと同部品で一貫。

## 4. 表示・入力項目

### 4.1 期間切替（`.rank-tabs`）
- **今週（既定）/ 先週 / 今月 / 通算**。切替でスコア窓・順位・表彰台・サマリーを再計算。
- 集計注記: 今週＝週次リセット（毎週月曜 0:00 起点）／先週＝前週確定／今月＝当月1日〜累計／通算＝作成来（リセットなし）。※週起点は初期値（TBD）。

### 4.2 あなたの順位サマリー
- **順位（N位）／全M人中／スコア**。**「▼ 自分の順位へ」**で全件リストの自分の行へスクロール（TOP3圏外でも即到達）。

### 4.3 表彰台（TOP3）
- 1位（中央・最も高い・金）／2位（銀）／3位（銅）。メダル＋アバター＋氏名（自分は「（あなた）」）＋スコア。

### 4.4 全件リスト
- **順位 / メダル（TOP3のみ）/ アバター（Lv付き）/ 氏名 / スコア（合計＋`EXP◯ ◆◯` 内訳）**。自分の行をハイライト。

## 6. データ・API（想定）

- **GET `/api/rankings?period=week|last|month|total`**: 会社内メンバーの期間スコア（獲得XP＋コイン）ランキング。各行＝ユーザー・Lv・スコア・XP/コイン内訳・順位。自分の順位/総人数も返す。
- 集計元＝**`Transaction/Activity`**（付与/消費履歴）を期間で集計。**同点はタイブレーク**（獲得XP優先→氏名順 等・初期案）。
- 大人数時は**ページング/仮想化＋自分の順位は別途取得**（TOP N＋自分の近傍）を想定（TBD）。**SP はランキング対象外**。

## 7. アクセシビリティ / 表示配慮

- 順位は色/メダルだけでなく数値（順位番号）で明示。自分の行は色＋「（あなた）」表記。
- CRTガラス上の明るい文字は on-dark トークン。スコア色（XP=ゴールド/コイン=アンバー）を区別。
- 期間タブは `role="tab"`/`aria-selected`。「自分の順位へ」のスクロール強調は `prefers-reduced-motion` 尊重（`animate` は装飾）。

## 8. 状態・バリエーション

| 状態 | 表示 |
| --- | --- |
| メンバー3人未満 | 表彰台は存在する順位のみ表示（欠けは出さない） |
| スコア0のメンバー | 下位に表示（0点）。同点はタイブレーク |
| 自分が1位 | サマリー「1位」＋表彰台中央に自分（「（あなた）」） |
| 大人数 | ページング/仮想化（TBD）。自分の順位へジャンプで到達 |
